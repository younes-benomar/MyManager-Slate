### 1. Preparation des fichiers
- Regroupez tous les fichiers du projet (html, css, js) dans un seul dossier.

### 2. Ouverture avec VS Code
- Lancez Visual Studio Code.
- Ouvrez le dossier contenant les fichiers (File > Open Folder).

### 3. Installation de l'extension
- Allez dans l'onglet Extensions (a gauche).
- Cherchez et installez l'extension nommée "Live Preview" (celle crée par Microsoft).

### 4. Lancement
- Ouvrez le fichier index.html dans l'éditeur.
- Faites un Clic Droit n'importe où dans le code.
- Sélectionnez l'option "Show Preview".

Le projet s'ouvrira directement dans une fenetre de prévisualisation.
